The QA40xPlot application consists of 8 different audio tests that use a QA402 or QA403 audio analyzer from QuantAsylum.

Test Description
Spectrum Capture the spectrum (V vs F) created by a sine wave. Analyze noise and harmonic distortion.
Intermodulation Capture the spectrum (V vs F) created by two sine waves. Analyze noise and intermodulation distortion.
Scope Capture the time signal (V vs mS) created by one or two sine waves. Much like an oscilloscope without the triggering.
THD vs Frequency Capture the spectrum created by a sine wave swept at multiple frequencies. Plot distortion vs frequency.
THD vs Amplitude Capture the spectrums created by a sine wave at the test frequency swept at multiple amplitudes. Plot distortion vs amplitude.
Response Capture the spectrums created by a sine wave swept at multiple frequencies. Plot output vs frequency.
Impedance Plot the impedance of the DUT at multiple swept frequencies. See below for more information on how to connect the dut.
Gain Plot the gain and phase of the DUT at multiple swept frequencies.

Common Options

Keyboard Accelerators

The left hand user interface entries on every tab are accessible by Alt-key combinations. Press and release the Alt key to see underlines appear on the prompts. Each underlined letter is a keyboard accelerator. So Alt-V, for example, usually lets you edit the voltage entry. Alt-R turns auto-ranging on and off, etc.

Graph Manipulation

Common to all tests are plots of values. You can change them easily by

  • Click a plot and drag the mouse to move the plot center point around.
  • Right-click a plot and drag the mouse to zoom in or out in the mouse direction.
  • Scroll the mouse wheel in a plot to zoom in or out
  • Click a plot axis and drag the mouse to reposition that axis.
  • Right-click a plot axis and drag the mouse to zoom in or out that axis.
  • Scroll the mouse wheel over a plot axis to zoom that axis in or out
  • --
  • Click inside the plot to freeze a cursor. Click again to unfreeze.

Amplitude Definitions

All tests have a choice as to how the QA40x generator voltage is determined.

Setting Description
Input Voltage The desired DUT input voltage. This is the QA40x generator output voltage.
Output Voltage Determine generator voltage to produce the desired DUT output. This does a low-voltage gain calculation first
Output Power Use the amplifier load setting (see Settings) to determine output voltage from power and then use gain to calculate the input/generator voltage

Attenuation Settings

QA40x attenuation is important for test device integrity. QA40xPlot tries very hard to use just enough attenuation for each test. Attenuation is always calculated based on both input channels (left and right) of the QA40x. If you only plan to test one channel we recommend you short the other QA40x input channel, which will guarantee that attenuation math ignores the shorted channel.

The free-running tests: Spectrum, Intermodulation and Scope, have manual or automatic attenuation. The other tests all automatically check gain then automatically determine attenuation to avoid QA40x overload in either channel.

The THD vs Frequency sweep calculates/uses one generator voltage that it then sweeps with. If you select an output amplitude that voltage ensures the maximum output of the sweep is that setting. The Input Voltage setting uses that voltage for the sweep.

The THD vs Amplitude test calculates distortion at each swept amplitude and varies generator voltage accordingly. Attenuation may increase as output voltage increases.

All 3 response tests (Response, Gain, Impedance) use a single generator voltage just like the THD vs Frequency test.

Cursor

All of the tests support a cursor. As you move your mouse in the plot a small box in the lower right shows the nearest data point. Some charts have a visual on-screen point, some don't.

Tab Get, Load, and Save

Each tab has a Get, a Load and a Save button in the left hand column at the bottom. When saved, the tab settings and plot data are saved to a compressed .plt file (by default named something.plt.zip). This file can be manually opened in Windows and the contents are a JSON text file. It can also, of course, be loaded.

When a saved tab is loaded two things happen. First, the plot data is displayed. Second, the original sweep settings are entered back into the user interface. So, the screen will look exactly as it did when the sweep/test was run. This lets you know exactly what settings you used for the test. It also allows for limited examination of the plot data.

When a saved tab is gotten (Get) the name will show up under the button with an L and R checkbox. Click the L and R to have the two channels show up on top of the current data. Click the color boxes below the L and R to change the plot line colors. Add more saved tabs by clicking the Get button again. Click the garbage icon to remove a saved tab. This will not delete the file on-disk.

Settings

The global Settings tab

There is one tab named "Settings" that shows the various global settings. These include colors, color themes, amplifier load, external gain block, and others.

Spectrum Test

This test sends a single sine wave out from the generator and plots the received frequency spectral data (value vs frequency). For a Sine wave this produces a skinny spike at the sine frequency surrounded by background noise and (usually harmonic) distortion spikes.

Response Test

This is a conventional frequency response test. The output is in dBV and can cover both channels or just one.

Options

  • Use Chirp - turn this on to use a chirp signal for the sweep. This is a more efficient way to sweep a frequency range. It suffers from reduced distortion and gain accuracy but gains almost continuous frequency coverage in a single packet. When using chirp, the right channel is used as reference if active.

Impedance Test

This does a frequency sweep and at each point calculates the effective impedance of the DUT. Data is stored as a complex value.

Options

  • Use Chirp - turn this on to use a chirp signal for the sweep. This is a more efficient way to sweep a frequency range. It suffers from reduced distortion and gain accuracy but gains almost continuous frequency coverage in a single packet.

Connections are:

  • DUT terminal A to ground
  • DUT terminal B to reference resistor terminal A and QA40x Left Input
  • reference resistor terminal B and generator output to QA40x Right Input

Gain Test

This does a traditional Bode Plot of Gain and Phase vs frequency. Here the right channel is used as a reference channel.

Options

  • Use Chirp - see above.

Connections are:

  • the QA40x right channel input to the DUT input port and generator output.
  • the QA40x left channel input to the DUT output port.
